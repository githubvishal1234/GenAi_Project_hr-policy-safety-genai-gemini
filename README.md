## HR Policy Safety GenAI (Gemini)

A secure GenAI chatbot built using Google Gemini API
to prevent prompt injection and protect confidential HR data.

### Features
- Gemini LLM integration
- Prompt-injection detection
- Polite refusal handling
- Gradio UI
- Automated attack testing

### Setup
1. Create `.env`
2. Add: GEMINI_API_KEY=your_key_here
3. pip install -r requirements.txt
4. python gradio_app.py
